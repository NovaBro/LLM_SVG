# Fine-Tuning LLM for SVG Generation

Setup instructions:

```bash
# Set up environment and download packages
./setup.sh -m env

# Download data from Kaggle
./setup.sh -m data
```

The files `EDA.ipynb` contains the EDA steps outlined in the report, `report.pdf`.
`starter_notebook_kaggle.ipynb` Contains code for our final method training setup. All other methods are recorded across the GitHub commit stages.
